# Contributing

Thank you for your interest in contributing to Awesome Spatial Omics Niche.

## Adding an Entry

Please ensure your pull request follows these guidelines:

- Search existing entries to avoid duplicates.
- Each entry should link to an actively maintained project, paper, or resource.
- Use the format: `- [Name](url) - Niche definition: description ending with a period.`
- Descriptions should be concise, objective, and explain what niche definition the method uses and why the resource is valuable.
- Include a journal citation where applicable: `(Journal, Year).`
- Add your entry to the appropriate section based on its niche definition type, in alphabetical order within that section.
- Check your spelling and grammar.
- Remove any trailing whitespace.

## Quality Standards

We aim to curate the **best** resources, not all of them. Before submitting, ask:

- Is this resource actively maintained?
- Does it offer something unique compared to existing entries?
- Does it clearly define or use a spatial niche concept?
- Would a researcher studying tissue microenvironments benefit from knowing about it?

## Updating the Companion Site

If you want to contribute deeper content (paper analyses, method comparisons, guides), please open an issue first to discuss scope and fit.

## Reporting Issues

If you find a broken link, outdated description, or inaccuracy, please open an issue or submit a pull request with the fix.
